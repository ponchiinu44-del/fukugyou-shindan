# fukugyou-shindan プロジェクトメモ

副業診断サイト（静的サイト）。`index.html` が診断本体、`trends.html` が収集データの一覧ページ。

## リサーチのルール（重要）

**Web検索などでリサーチを行ったら、毎回その結果を `research/YYYY-MM-DD.md` に保存すること。**

保存内容には以下を必ず含める：
- 実行した検索クエリ一覧
- ジャンルごとの要点（SNSトレンド／時事／ネタ案など）
- 参照元のURL（ヒットした記事リンク）

対象ジャンルは原則 **①占い ②副業 ③アフィリエイト** の3つ（X・Instagram・YouTube 中心）。
同日に複数回リサーチした場合は同じ日付ファイルに追記する。

## ディレクトリ

- `index.html` … 15問の副業診断。結果カードに「SNSでの伸ばし方（トレンド）」を表示。
- `trends.html` … 診断不要で全ジャンルのトレンドと「今日のネタ」を一覧表示。データは各ファイル内の配列で管理。
- `research/` … リサーチ記録（日付別 Markdown）。
