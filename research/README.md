# SNSリサーチ蓄積フォルダ

副業ジャンルで「伸びている投稿の型・テーマ」を日次で集め、自分の X / Threads 投稿ネタに活かすための記録です。

## 運用ルール

- ファイル名：`research/YYYY-MM-DD.md`（例：`2026-06-22.md`）
- 毎日このセッションで「**今日の分集めて**」と送ると、X・Instagram・YouTube の伸びている投稿テーマを各10本、出典付きで収集してこのフォルダに追加します。
- フォーマットは**箇条書き**。各項目に「伸びてる理由」と、**そのまま使える例文**を添えます（`〇〇` は自分の言葉に置き換え可）。
- 末尾に「今日のひとことまとめ」と、その日いちばんおすすめの1投稿例文を載せます。

## 注記

X・Instagram は認証なしで個別投稿の数値（いいね・再生数）を取得できないため、信頼できる記事・運用者の発信から「いま伸びている投稿の型・テーマ」を集約しています。

## 履歴
