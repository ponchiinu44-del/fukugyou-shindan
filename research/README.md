# SNSリサーチ蓄積フォルダ

SNSで「伸びている投稿の型・テーマ」を日次で集め、自分の X / Threads 投稿ネタに活かすための記録です。

## フォルダ構成（ジャンル別）

- `research/fukugyou/YYYY-MM-DD.md` — 副業ジャンル
- `research/affiliate/YYYY-MM-DD.md` — アフィリエイトジャンル

## 運用ルール

- 毎日このセッションで「**副業の今日の分集めて**」「**アフィリの今日の分集めて**」と送ると、該当ジャンルの X・Instagram・YouTube の伸びている投稿テーマを各10本、出典付きで収集してフォルダに追加します。（ジャンル指定なしの「今日の分集めて」は副業＝デフォルト）
- フォーマットは**箇条書き**。各項目に「伸びてる理由」と、**そのまま使える例文**を添えます（`〇〇` は自分の言葉に置き換え可）。
- 末尾に「今日のひとことまとめ」と、その日いちばんおすすめの1投稿例文を載せます。

## 注記

X・Instagram は認証なしで個別投稿の数値（いいね・再生数）を取得できないため、信頼できる記事・運用者の発信から「いま伸びている投稿の型・テーマ」を集約しています。アフィリエイトは PR/広告表記（ステマ規制）と各SNS・ASPの規約の確認が前提です。
